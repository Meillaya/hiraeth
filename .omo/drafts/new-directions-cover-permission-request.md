# Draft request: New Directions cover thumbnail permission

This document is a draft/request only. It is not granted permission, does not record consent, and must not be treated as authorization to cache, display, or reuse New Directions cover images.

Hello New Directions Permissions Team,

Hiraeth is a small, non-commercial catalog prototype for browsing curated independent publisher metadata. We are requesting written permission to cache and display reduced-size cover thumbnails for New Directions titles already represented in our catalog from source-backed factual bibliographic metadata.

Requested use:

- Cache cover images only as small web thumbnails for catalog discovery pages and title detail pages.
- Display clear attribution to New Directions and link each title back to the relevant official New Directions page when available.
- Use the thumbnails only to identify and browse books; Hiraeth will not provide checkout, ordering, sales, inventory, price tracking, or commercial resale of the images or books.
- Avoid using cover images in advertising, merchandise, downloadable image packs, or any separate commercial product.

If New Directions grants written permission, we would retain the written terms with the provider policy and limit caching to the approved scope. Until that written permission is received and recorded, New Directions covers remain disabled in Hiraeth: no local cached thumbnails and no public cover-image rendering from the app.

Takedown/removal process:

- New Directions may request removal, correction, or cache purge for any title or image at any time.
- Hiraeth will remove the requested cached thumbnail(s), clear generated derivatives, and update the provider policy/fixture notes to prevent re-import.
- Removal requests can include a title, ISBN, URL, or other identifying information; no formal notice format is required for prompt removal.

Thank you for considering this request. We would be happy to adjust the scope, attribution wording, or takedown process to match New Directions' requirements.
